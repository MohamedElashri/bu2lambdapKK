# Archive

This directory holds superseded historical or reference material that is
intentionally kept out of the active reproducible workflow.

What belongs here:

- frozen reference workflows that were replaced by the top-level
  `analysis/Snakefile`
- historical notes or assets we want to retain but not treat as active
- material that should stay readable but should not stay in the main source or
  standalone study trees

What does not belong here:

- active study code used by `analysis/Snakefile`
- standalone exploratory studies that we still want visible and attributable
- presentation workflows
- generated outputs from the current workflow

Current contents:

- `reference_workflows/`
  - archived standalone study-local Snakefiles superseded by the top-level DAG
