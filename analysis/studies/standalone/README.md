# Standalone Studies

This directory holds exploratory or documentation-only studies that are kept on
purpose, but are not part of the active top-level Snakemake workflow.

Use this area for:

- one-off decision-support studies
- provenance studies whose conclusions are still worth keeping nearby
- documentation-only studies that explain assumptions used elsewhere

Do not use this area for:

- active workflow stages owned by `analysis/Snakefile`
- presentation-only plotting workflows
- superseded frozen reference workflows that should instead live in
  `analysis/archive/`
